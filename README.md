# 魔塔游戏 (Godot 4.4)

## 项目简介

这是一款基于Godot 4.4引擎开发的经典魔塔游戏。玩家需要在一座多层的塔内探索，收集钥匙、打败怪物、获取宝物，最终到达塔顶完成游戏目标。

## 特色功能

- 回合制战斗系统
- 多样化的怪物和道具
- 层层递进的难度设计
- 资源管理（生命值、攻击力、防御力、钥匙等）
- 解谜元素

## 开发环境

- 引擎: Godot 4.4
- 编程语言: GDScript
- 版本控制: Git

## 项目结构

```
/project
  /assets        - 游戏资源（精灵、音频、字体等）
  /scenes        - 游戏场景
  /scripts       - 游戏脚本
  /resources     - 游戏数据资源
```

## 如何开始

1. 确保已安装Godot 4.4或更高版本
2. 克隆或下载本项目
3. 使用Godot引擎打开项目文件夹
4. 点击运行按钮开始游戏

## 游戏控制

- 方向键或WASD: 移动角色
- E或空格键: 交互
- ESC: 打开游戏菜单
